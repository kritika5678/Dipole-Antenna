# Dipole-Antenna
**PROCEDURE**

1) In the experiment we drew two cylinders one outer (Cylinder 2) and one inner (Cylinder 1).

2) Cylinder 1 has the material set as Copper and cylinder 2 had the material Air.

3) Cylinder 1 has radius 1 mm and height 15 mm. It also has a duplicate cylinder named Cylinder 1_1.
4) Cylinder 2 has radius 50 mm and height 400 mm. It is used as radiation box.
